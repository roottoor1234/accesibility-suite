Καλώς ήρθατε στο Demo

Αυτή η σελίδα επιδεικνύει πώς να ενσωματώσετε το widget προσβασιμότητας σε οποιαδήποτε ιστοσελίδα χωρίς να χρειάζεται React ή άλλα frameworks. Το widget είναι πλήρως αυτόνομο και λειτουργεί με απλό HTML!

💡 Συμβουλή: Κάντε κλικ στο μπλε κουμπί προσβασιμότητας στο κάτω δεξιά για να ανοίξει ο πίνακας ελέγχου.

📋 Διαθέσιμα Χαρακτηριστικά

Το widget προσφέρει 11 πλήρως λειτουργικά εργαλεία προσβασιμότητας:

📏 Μέγεθος Κειμένου

Αυξήστε ή μειώστε το μέγεθος του κειμένου από 80% έως 200%

🎨 Υψηλή Αντίθεση

Ενεργοποιήστε τη λειτουργία μαύρου φόντου με λευκό κείμενο

⏸️ Παύση Κινήσεων

Απενεργοποιήστε όλες τις κινήσεις και τις μεταβάσεις

🔗 Επισήμανση Συνδέσμων

Κάντε τους συνδέσμους πιο ορατούς με κίτρινο φόντο

👁️ Οδηγός Ανάγνωσης

Ένας οπτικός κανόνας που ακολουθεί το ποντίκι σας

📖 Γραμματοσειρά Δυσλεξίας

Γραμματοσειρά φιλική προς άτομα με δυσλεξία

🖼️ Απόκρυψη Εικόνων

Απόκρυψη εικόνων για περισσότερη εστίαση στο κείμενο

📐 Ύψος Γραμμής

Προσαρμόστε το κενό μεταξύ των γραμμών (80-200%)

↔️ Απόσταση Γραμμάτων

Αυξήστε την απόσταση μεταξύ των γραμμάτων

🖱️ Μέγεθος Δείκτη

Μεγαλώστε τον κέρσορα του ποντικιού (100-300%)

🔊 Ανάγνωση Σελίδας

Ακούστε ολόκληρο το περιεχόμενο της σελίδας δυνατά

🚀 Οδηγός Ενσωμάτωσης

Για να χρησιμοποιήσετε το widget στη δική σας ιστοσελίδα, απλά προσθέστε αυτά τα 2 αρχεία:

Βήμα 1: Προσθέστε το CSS

<!-- Φόρτωση CSS του widget -->
<link rel="stylesheet" href="https://your-cdn.com/a11y-widget.css">

Βήμα 2: Προσθέστε το JavaScript

<!-- Φόρτωση JS του widget -->
<script src="https://your-cdn.com/a11y-widget.js"></script>

Βήμα 3: Αρχικοποίηση (Προαιρετικό)

Το widget αρχικοποιείται αυτόματα, αλλά μπορείτε να το ελέγξετε χειροκίνητα:

<script>
  // Αρχικοποίηση με προσαρμοσμένη θέση
  window.initA11yWidget({
    position: 'bottom-right' // ή 'bottom-left', 'top-right', 'top-left'
  });
</script>
💾 Αυτόματη Αποθήκευση: Όλες οι ρυθμίσεις αποθηκεύονται αυτόματα στο localStorage και παραμένουν ενεργές σε όλες τις επισκέψεις!

📦 Αρχεία Build

Μετά το build, θα δημιουργηθούν τα εξής αρχεία στον φάκελο dist-widget/:

  • ✅ a11y-widget.js (~492KB) - Το κύριο JavaScript bundle
  • ✅ a11y-widget.css (~20KB) - Όλα τα styles του widget

Για να δημιουργήσετε τα αρχεία, εκτελέστε:

npm run build:widget

🌐 Συμβατότητα Browser

Το widget υποστηρίζει όλους τους σύγχρονους browsers:

  • Chrome / Edge 90+
  • Firefox 88+
  • Safari 14+
  • Mobile browsers με υποστήριξη Web Speech API

📝 Σημειώσεις

  • Το widget είναι πλήρως αυτόνομο - δεν χρειάζεται εγκατάσταση dependencies
  • Όλες οι ρυθμίσεις αποθηκεύονται τοπικά στον browser του χρήστη
  • Το widget σέβεται τη ρύθμιση prefers-reduced-motion του συστήματος
  • Πλήρως προσβάσιμο με πληκτρολόγιο (Tab, Esc, κλπ)
  • Δεν επηρεάζει το υπόλοιπο περιεχόμενο της σελίδας
